# TODOトラッカー with タイムトラッキング

タスク管理と作業時間記録を統合したCLIアプリケーション

## 概要

Python3とSQLiteで実装したタスク管理ツールです．タスクの登録・管理に加えて，各タスクの作業時間を記録・集計できます．

## 機能

- **タスク管理**
  - タスクの登録・完了・削除
  - 複数カテゴリの設定（多対多リレーション）
  - 期日の設定
  
- **タスク一覧表示**
  - 未完了/完了済みの切り替え
  - カテゴリでの絞り込み
  - 期限順/登録順の並び替え

- **タイムトラッキング**
  - 作業時間の記録
  - タスク別累計時間の表示
  - 日別統計の表示
  - カテゴリ別統計の表示
  - 完了率の計算

## 技術スタック

- **言語**: Python 3.7+
- **データベース**: SQLite3
- **依存関係**: なし（標準ライブラリのみ）

## データベース設計

### テーブル構成

- `tasks`: タスク情報
- `categories`: カテゴリ情報
- `task_categories`: タスクとカテゴリの中間テーブル（多対多）
- `time_records`: 作業時間記録

### 実装されているSQL操作

- `INSERT`: タスク・時間記録の登録
- `SELECT WHERE`: 条件抽出
- `UPDATE`: 完了フラグ更新
- `DELETE`: タスク削除
- `JOIN`: 複数テーブルの結合
- `GROUP BY + SUM`: 集計処理

## 使い方

### メニュー操作

アプリを起動すると対話式メニューが表示されます．

```

=== TODOトラッカー ===

1. タスク登録
2. タスク一覧表示
3. 作業時間記録
4. タスク完了
5. タスク削除
6. 統計表示
7. 終了

```

数字を入力して機能を選択し，プロンプトに従って操作します．

### タスク登録の例

```

選択してください: 1

[タスク登録]

タスク名: データベース課題

詳細（任意）: レポート作成

期日（YYYY-MM-DD，任意）: 2026-01-15

カテゴリ（カンマ区切り，任意）: 大学, DB, 緊急

タスクを登録しました（ID: 1）

```

### 作業時間記録の例

```

選択してください: 3

[作業時間記録]

タスクID: 1

作業時間（分）: 120

120分を記録しました

```
